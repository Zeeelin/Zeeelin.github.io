Description of the Project

Make a Mind is a virtual reality application aiming to implement some traditional psychological
therapies into the virtual world, providing more affordable and accessible psychological therapies
for people in need. It contains four minigames from music boxes to destruction therapy to exposure
therapy and memory palace. The application is developed in Unreal Engine 5.

Role in the Project

My role in this project is mostly creating the memory palace-inspired mini-game in Unreal Engine 5 using blueprints.
While most 3D assets used in this software come from Sketchfab and Unreal Asset Store, I was also able to explore
the particle system in UE5 as well as a Blender to work with the models, aiming to create a cozy feeling with the
in-game environment. The inspiration for creating this memory palace-themed mini-game is that memory has always intrigued
me during my undergrad study in cognitive science, and I was curious whether practicing this technique in virtual
reality could help people utilize it.

Here is a short description of how the memory palace works: The tranquil study room in this virtual palace
features seven objects attached to pieces of furniture arranged in clockwise order. Users are encouraged to
explore the room, become familiar with the objects, and remember them in the exact same sequence each time they
revisit the palace. Spatial association helps players enhance memory retention, and a tutorial is also included
to guide users around.

Application/Software Used: Unreal Engine 5, Blender, Jira, Photoshop, Github
