Description of the Project

The Seed was a game created during the 48 hours of Global Game Jam 2023. It is a
point-and-click narrative game inspired by the game Bad Dream Coma. The game tells
the story of a rookie detective investigating a serial killer case that targets the
police and related personnel.

Role in the Project

I teamed up with one other artist to work together during this game jam. I worked mostly
as a programmer and helped with some of the UI art. The game engine we used was Unity, and
we designed the game's narrative together. Unfortunately, we were not able to finish the game
in 48 hours, given the scope of the script. It was a rush for both the artist and me as the
programmer. It was the first game jam in which I was the only programmer on the team. I was able
to learn a lot by consulting the local Jam host, the programming lead at WolverineSoft Studio.
He taught me a lot of shortcuts in Unity, so I do believe that if I decide to recreate this game,
I could make it much more polished in a shorter time period. Overall, this was a valuable experience
for me.
