Role in the Project

I worked mostly on the character assets as well as some user interface elements and backgrounds
for narratives. The character assets include the different upgrades for each of the four characters
I was in charge of. (As shown below in the sample work section) I also created the main and pause menus
as well as two narratives cutscene for the game. The theme and narrative of this game were created and
decided collaboratively by the art team. I used both Photoshop and Procreate to draw the asset for this game.
